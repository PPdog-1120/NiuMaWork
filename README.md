# 加班计时器 (Overtime Tracker) v1.3

一款 Android 应用，帮助员工记录加班时长，支持弹性工作制，区分工作日与周末/节假日，提供补录与修改能力。

## 功能特性

### v1.3

- **弹性工作制**：支持配置弹性最早/最晚上班时间，自动归一化打卡时间
- **午休配置**：工作日和周末/节假日独立配置午休起止时间
- **晚饭扣除**：工作日和周末/节假日独立配置晚饭扣除时长
- **加班计算**：精确到分钟，不取整
  - 工作日：`加班 = max(0, 实际下班 - 应下班 - 晚饭扣除)`
  - 周末/节假日：自动扣除跨越的午休和晚饭时段
- **请假系统**：支持全天/半天/自定义时长请假
- **净加班**：`净加班 = 总加班 - 总请假`，可为负数
- **补录与修改**：支持手动补录打卡和请假记录，标记来源
- **跨天打卡**：适用于夜班场景
- **历史记录**：日历视图 + 列表视图，支持按月/周筛选
- **统计分析**：今日/本周/本月/本年累计，净加班统计，加班构成分析
- **桌面小组件**：4×1 标准版、2×1 紧凑版、2×2 方形版

## 计算公式

### 工作日

```
归一化上班 = clamp(实际打卡, 弹性最早, 弹性最晚)
应下班     = 归一化上班 + 标准工时 + 午休时长
加班时长   = max(0, 实际下班 - 应下班 - 晚饭扣除)
```

### 周末 / 节假日

```
总时长 = 结束 - 开始
IF 跨午休:  总时长 -= 午休时长
IF 跨晚饭:  总时长 -= 晚饭扣除时长
加班时长 = max(0, 总时长)
```

### 净加班

```
净加班时长 = 总加班时长 - 总请假时长
```
